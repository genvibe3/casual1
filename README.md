# 🏃‍♀️ 한강 러닝 - 헌팅남 피하기 게임

서울 한강공원을 배경으로 하는 횡스크롤 러닝 게임입니다. 젊음을 유지하면서 헌팅남들을 피하고 최대한 오래 달려보세요!

## 🎮 게임 소개

20대 여성이 한강공원에서 달리는 동안 다양한 헌팅남들을 피하면서 코인을 모으고 콜라겐 캡슐로 나이를 회복하는 게임입니다.

### 주요 특징

- 🌆 **서울 랜드마크 배경**: 롯데월드타워, N서울타워, 숭례문 등 실제 서울의 모습
- 👤 **나이 시스템**: 시간이 지나면 자연스럽게 나이가 들어 외형이 변화
- 💊 **콜라겐 캡슐**: 2단 점프로 획득하여 나이를 회복
- 🚗 **다양한 헌팅남**: 걷는 사람, 자전거, 차 등 다양한 방식으로 접근
- 💬 **동적 대화 시스템**: 거리에 따라 말투가 변하는 헌팅남들
- 🏃 **일반 시민**: 런닝하는 일반 시민들도 등장
- 🎨 **자연스러운 애니메이션**: 머리카락 물리, 옷 움직임 등

## 🎯 게임 방법

### 조작법
- **스페이스바** 또는 **↑** - 점프
- **공중에서 한번 더!** - 2단 점프 (회전 애니메이션)
- **↓** - 슬라이드
- **ESC** - 일시정지

### 게임 규칙
1. 20세에서 시작하여 시간이 지나면 자연스럽게 나이가 듭니다
2. 80세가 되면 게임 오버!
3. 헌팅남들과 부딪히면 게임 오버
4. 코인(₩)을 모아 점수를 올리세요
5. 콜라겐 캡슐(분홍색)을 먹으면 나이가 15세 회복됩니다 (2단 점프로만 획득 가능)
6. 시간이 지날수록 게임 속도가 빨라집니다

### 헌팅남 종류
- **걷는 헌팅남**: 다양한 헤어스타일과 의상 (캐주얼, 정장, 후드티, 운동복, 재킷)
- **자전거 헌팅남**: 더 빠른 속도로 접근
- **차 타고 오는 헌팅남**: 가장 빠른 속도

### 대화 시스템
헌팅남들은 플레이어와의 거리에 따라 다른 말을 합니다:
- **멀 때**: "안녕하세요~", "저기요!", "잠깐만요~"
- **중간 거리**: "오늘 멋진데요?", "인스타 해요?", "혼자 달려요?"
- **가까이**: "사랑해요! ❤️", "저랑 사귈래요?", "데이트 신청합니다!"

## 🎨 캐릭터 외형 변화

나이에 따라 캐릭터의 외형이 자연스럽게 변합니다:
- **20대**: 은회색 머리, 건강한 피부
- **40대**: 머리색이 점점 하얗게, 주름이 생기기 시작
- **60대 이상**: 백발, 허리가 굽기 시작

## 🛠️ 기술 스택

- **Phaser 3** (v3.70.0): 메인 게임 엔진
- **Matter.js**: 물리 엔진 (Phaser 내장)
- **JavaScript ES6**: 게임 로직
- **Press Start 2P**: 레트로 픽셀 폰트
- **LocalStorage**: 최고 기록 저장

## ✨ 특수 효과

### 파티클 시스템
- 점프 효과 (은색/분홍색 파티클 버스트)
- 코인 획득 (금색 스파클)
- 콜라겐 획득 (분홍색 폭발)
- 충돌 (빨강/주황/노랑 폭발)

### 카메라 효과
- **카메라 쉐이크**: 충돌 시 300ms 진동
- **타임 슬로우 모션**: 콜라겐 획득 시 2초간 0.3배속

### 포스트 프로세싱
- **CRT 스캔라인**: 레트로 CRT 모니터 효과
- **비네팅**: 화면 가장자리 어둡게 처리
- **픽셀 아트 렌더링**: crisp-edges 필터링

## 🚀 실행 방법

1. 저장소를 클론합니다:
```bash
git clone [your-repository-url]
```

2. `game` 폴더로 이동합니다:
```bash
cd 캐쥬얼게임1/game
```

3. `index.html` 파일을 브라우저에서 엽니다

또는 웹 서버를 실행합니다:
```bash
# Python 3
python -m http.server 8000

# 그 다음 브라우저에서 http://localhost:8000 접속
```
